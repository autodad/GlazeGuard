Bill of Materials (BOM): GlazeGuard Prototype V1.0

This list covers the components required to build and test a single PCPT (Phase-Change Power Traction) unit.

1. Core Components (The "Ice-Drive")
Item	            Description	                    Source / Suggestion
Expansion Medium	Distilled Water	                Supermarket or Pharmacy.
Reinforcement	    Nanocellulose (CNC or CNF)	    Laboratory supplier or DIY (Plant-based extraction).
Housing Material	Flexible Filament / Resin	    TPU (95A for housing, 85A for column).
Diaphragm	        Bistable Spring Steel Disc	    10-15mm dome switches (from heavy-duty buttons).


2. Traction Elements
Item	            Description	                    Source / Suggestion
Traction Stud	    Tungsten Carbide Tip	        Standard tire studs or replacement trekking pole tips.
Buckling Column	    Custom 3D-printed S-curve	    Printed in TPU (part of the Upper Assembly).


3. Sealing & Assembly
Item	            Description	                    Source / Suggestion
Sealant	            Cyanoacrylate or Epoxy	        High-strength waterproof adhesive (e.g., Gorilla Glue).
Syringe	            1ml - 5ml Syringe	            For precise injection of the NC-water mixture.
Gasket (Optional)	Tiny O-rings	                Nitrile rubber (3-5mm diameter) if using a mechanical seal.


4. Testing Equipment (The "Home Lab")
Item	            Description	                    Source / Suggestion
Thermal Source	    Standard Freezer	            Needs to reach -18°C for rapid testing.
Force Gauge	        Digital Luggage Scale	        To measure the Buckling Point (Pcr​) in kg/N.
Calipers	        Digital Caliper	                To measure the 9% expansion displacement.


# Implementation Note for Developers
To keep the first prototype simple, the Lower Housing can be 3D-printed in a cup-shape. After filling it with the NC-water mixture, the Bistable Diaphragm should be glued directly to the rim to create a hermetic seal before attaching the Buckling Column and Stud.
